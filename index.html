Welcome to ANSA project webpages

The Automated Network Simulation and Analysis (ANSA) is a long-term project carried by researchers and students at the Brno University of Technology (namely at Faculty of Information Technology). It is dedicated to develope a variety of simulation models compatible with RFC specifications or referential implementations, which extend wired IP network framework INET. Subsequently, these tools would allow formal analysis of real networks and their configurations. ANSAINET may be publicly used as the routing/switching baseline for further research initiatives, i.e., in simulations for proving (or disproving) certain aspects of technologies and/or related protocols. One of the main goals is to integrate certain ANSAINET features into INET.

Installation and Branches

Installation is pretty simple, follow the cookbook:

  1. Just clone or download repository's main branch. This branch already contains the newest version of supported INET (currently INET 3.3.0 in folder /src/inet) together with ANSAINET extension /src/ansa.
  2. Import the project into OMNeT++ (it is named ansainet-*, which allows you to have also inet project in the same worspace).
  3. Compile the project; it will take just a little more time than traditional INET.
  4. Play with ANSAINET; run fingerprinted scenarios in /examples/ansa folder and check the source codes in /src/ansa folder.

The most up-to-date branch is ansainet-3.3.0. However, several branches still exist (thou unmainted) in order to provide functionality to deprecated version of OMNeT++ simulator. Among these branches are:

  • ansainet-3.2.1 for OMNeT++ 5.0
  • ansainet-3.0 for OMNeT++ 4.6
  • ansainet-2.2 for OMNeT++ 4.6
  • ansainet-2.1 for OMNeT++ 4.6
  • ansainet-2.0 for OMNeT++ 4.5
